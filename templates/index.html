One of Udacity's capstone projects is to use Convolutional Neural Networks to write and algorithm for a dog identification app. The goal is to classify dogs according to their breed. Code for some of the tasks were included in the notebook. This web app will accept any user-supplied image as input. If a dog is detected in the image, it will provide an estimate of the dog's breed. If a human is detected, it will provide an estimate of the dog breed that the human most closely resembles.
